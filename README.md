
# 🧮 Python Web Calculator

This is a simple yet stylish calculator built using:

- ⚙️ Python + Flask (backend)
- 🖥️ HTML, CSS, and JavaScript (frontend)
- 🎨 Creative design with display and keys 

## 🚀 How to Run

### 1. Install dependencies:
```
pip install -r requirements.txt
```

### 2. Run the app:
```
python app.py
```

### 3. Open in browser:
```
http://127.0.0.1:5000/
```
